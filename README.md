SmartCart Customer Segmentation System
Developed an AI/ML-based customer segmentation system using Unsupervised Learning
Performed Data Preprocessing and One-Hot Encoding for categorical features
Applied Feature Engineering to improve model performance and customer analysis
Used the Elbow Method and Silhouette Score to determine the optimal number of clusters
Implemented and compared different clustering algorithms
Found Agglomerative Clustering performed best for customer grouping
Analysed customer behaviour patterns through cluster comparison
Generated insights for personalised marketing and customer retention strategies
Built using Python, Pandas, Scikit-learn, Matplotlib, and Seaborn
